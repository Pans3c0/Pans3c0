### 👨‍💻 Sobre Mí

Soy un desarrollador de software con un perfil híbrido. Por un lado, domino la lógica de negocio profunda de los sistemas **ERP (Sage X3, Navision/Business Central)**, creando soluciones para el sector industrial y logístico. Por otro, construyo aplicaciones web modernas y escalables utilizando **Spring Boot, React y Docker**.

* 🏢 **En el trabajo:** Desarrollo pantallas, modelos de importación y ofrezco soporte técnico especializado en entornos críticos de gestión (Sage X3 / BC).
* 🎓 **Reciente Graduado FP:** Mi proyecto final es un **Todo List Multiservicio** usando Spring Boot (APIs), React y desplegado en contenedores Docker.
* 🖥️ **Homelab & DevOps:** Administro mi propio servidor de infraestructura (Proxmox sobre un Intel N100), donde despliego mis proyectos, gestiono contenedores y monitorizo la red y los recursos.
